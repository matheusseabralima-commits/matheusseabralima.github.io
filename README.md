## Como publicar no GitHub Pages

1. Crie um repositório com o nome exato: `SEU_USUARIO.github.io`
2. Faça upload dos arquivos (`index.html` + `foto.jpeg`)
3. Vá em **Settings → Pages → Source: main branch / root**
4. Pronto! Em ~1 minuto o site estará no ar.
